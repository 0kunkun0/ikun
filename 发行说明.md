## 概述
- **ikun库是一个跨平台的C++编程工具库, 支持Windows, Linux和macOS系统, 遵循GPL 3.0许可证**, 此版本是**7.1.2正式版**
 - 提供了丰富的功能, 包括但不限于:
 *文件操作*(files.hpp)
 *数学运算*(maths.hpp)
 *随机数生成*(random.hpp)
 *时间处理*(times.hpp)
 *错误处理*(ikun_stderr.hpp)
 *控制台颜色输出*(console_color.hpp)
 *输入输出整合*(io.hpp)
 *高精度整数运算*(high_precision_digit.hpp)
 *C++标准库整合*(stdc++lib.hpp)等
 - 允许在开源的情况下向ikun库添加扩展功能, 但**必须遵循GPL 3.0许可证规定, 且所有衍生版本必须按照许可证规定开源**

- 开发工具方面, 对开发者提供错误分析器(仅限本库的代码)`ikun_error_analyzer.cpp`和库/项目管理工具`ikun_core.cpp`(编译后应为`ikun(.exe)`)

- 法律方面, ikun库遵循GPL 3.0许可证, **禁止使用本库进行商业闭源用途**

- 备注: 由于本库使用极其新的C++标准(**C++23**), 所以**请保证你的编译器支持C++23**, 由于本库不存在使用`std::cout`输出的代码, 所以**移植到低版本编译器非常困难**
---
## 新增功能
- 添加`sysapi.hpp`, 提供对操作系统API和与平台相关代码的封装, 使其能跨平台使用
 - 备注: 仅支持Windows API和POSIX API

- 添加`ikun_error_analyzerr.cpp`中对`sysapi.hpp`的错误处理

- 添加`ikun_using_libstdc++.hpp`, `ikun_using_libc++.hpp`和`ikun_using_msvc.hpp`模块, 功能为选择性的`using std::xxx;`的封装, 用于在`using namespace std;`的兼容性问题和不`using namespace std;`的便捷性中间的平衡(选择性地去除了`algorithm`库的minmax函数, 防止与Windows API冲突)

- `high_precision_digit.hpp`中, 添加获取和估算`bigint`对象大小的函数`object.get_size()`和`est_bigint_memory(const bigint& n)`

- `ikun_core.cpp`中, 添加`compile()`函数和`--compile`选项, 可用于快速编译现有的C++源文件
